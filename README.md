# CattleID

This repository acts as a public access for the source code and implementation commented and detailed in [ARTICLE].
This project was developed by students and researchers at Universidade federal do Paraná (UFPR) and Instituto de Desenvolvimento Rural do Paraná (IDR).

## Means of use

This code can be used by obtaining a graph structure, found in the "structures" folder, or a similar might be used.
After obtaining two graphs you wish to compare, use "match_recursive" to compare both graphs utilizing the method describe in the paper.
